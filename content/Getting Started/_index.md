DFIR-OGRE provides two installation methods:

- A [Docker installation](/getting-started/docker_installation/ "Docker installation"). It is the recommended way to use DFIR-OGRE as it allows for fine control over ressources usage.

- A [manual installation](/getting-started/manual_installation/ "Manual installation"). It is recommended for debugging or custom plugin development.
